## Homework creation tool for MATLAB class teaching assistants

<figure class="image">
  <img src="./assets/tcc.png" alt="Test case compiler">
  <figcaption style="text-align: center; margin-block: 20px;">Test Case Compiler user interface</figcaption>
</figure>

The Test Case Compiler is a software tool used by teaching assistants for the
course CS 1371 (Computing for Engineers) at Georgia Tech. Its
purpose is to package homework problems and create rubrics for
automatically grading them.
